# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records for the EMC Workspace platform and tenant implementations.

## Status Legend

- **Proposed:** Under review. Not yet implemented or may still change.
- **Accepted:** Approved and implementation is authorized.
- **Deprecated:** No longer recommended for new work.
- **Superseded:** Replaced by a newer ADR.
